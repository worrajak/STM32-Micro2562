# STM32Micro2562
จัดทำเพื่อให้เป็นความรู้ในการใช้งาน STM32F103C8T6 กับการใช้งานกับอุปกรณ์ต่างๆ 

```
LED1=PC_13  SERIAL_TX=PA_2  I2C_SCL=PB_8  SPI_MOSI=PA_7  PWM_OUT=PB_3
            SERIAL_RX=PA_3  I2C_SDA=PB_9  SPI_MISO=PA_6
                                          SPI_SCK =PA_5
                                          SPI_CS  =PB_6
```
